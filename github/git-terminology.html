git bash - the tool that allows you to access git via command line git init - initializes to start a new project/start an empty repository git clone - copy files to your computer git status - lists all new or modified files you have that are ready to be committed; use to check the status of the working directory git add - add all files to the staging area git add filename - add specific file to the staging area git commit - adds your file changes permanently to the version history (in the files you've added) git log - allows you to see commits git pull - allows you to pull down newest updates from the server before committing
